# Development Journal (Index)

## Active journal file
- `journal/THIS-MONTH.md` (rename to `YYYY-MM.md` when you start using it)

## Rule
- Journal is append-only: do not rewrite old entries.
- Write new entries into the active monthly file.

## Journal entry template
## YYYY-MM-DD
- [Area/Subarea][Optional Tags] What changed
  - Why it changed
  - Key files touched
  - Verification (docs/tools) + assumptions
  - Regression checks run (PASS/FAIL)
